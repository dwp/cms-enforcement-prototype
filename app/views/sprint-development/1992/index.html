Statement -- Statement MoJ button off-pattern

Description:

The current design for filtering the page has usability and accessibility issues. It is an off-pattern modification of the MoJ "button menu" which was intended to group and hide multiple disparate buttons, each one clearly labeled with the action it performs. Instead the current implementation labels the button with category descriptions without providing information about the expected funtionality.

The GOVUK Select components has better usability and accessibility for a filtering function.

Changes:

  • Replace 'MoJ button menu' with 'GOVUK select'
