Statement -- Statement MoJ button off-pattern

Description:

The current filtering component using the MoJ "Button menu" needs replacing for a more usable and accessible one. The "Button menu" component is designed to cluster and hide multiple disparate buttons which perform on-click actions and have lables that explain their outcome.

The current implementation however uses button labels that describe table data type instead. It does not describe an action (filtering) or the functionality (action is triggered when clicking the label).

The GOVUK Select components has better usability and accessibility for a filtering function.

Changes made:

  • Replace 'MoJ button menu' with 'GOVUK select'
