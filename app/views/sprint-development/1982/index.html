Pre-2016 calculation limitations

Description:

Users will need to check CMS012 to see if charges are applicable for any calculations that pre date 2016. This is because the service type will be returned blank in the SR for anything pre 2016, as it was only added to the SR post 2016. Therefore VPF can not determine whether the charges apply or not.
