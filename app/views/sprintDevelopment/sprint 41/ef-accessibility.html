Accessibility statement for the View parent finances service

This accessibility statement applies to View parent finances. This is an agent-facing service used in the Department of Work and Pensions (DWP). No citizens use this service and it is not available on GOV.UK.

View parent finances allows authorised users to view a range of data items which are held in other DWP systems.

Use of the service is secure, controlled by access permissions set by DWP.

Using the service

This service is run by the Department for Work and Pensions (DWP). We want as many child maintenance case workers as possible to be able to use this service. For example, that means you should be able to:

  • change colours, contrast levels and fonts using browser or device settings
  • zoom in up to 400% without the text spilling off the screen
  • navigate most of the service using a keyboard or speech recognition software
  • listen to most of the service using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

We’ve also made the text in the service as simple as possible to understand.

AbilityNet (opens in a new window) has advice on making your device easier to use if you have a disability.

Reporting accessibility problems with this service

We’re always looking to improve the accessibility of this service. If you find any problems that are not listed on this page or think we’re not meeting accessibility requirements, you can give feedback.

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 (the ‘accessibility regulations’).

If you’re not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS).

If you are in Northern Ireland contact the Equalities Commission for Northern Ireland (ECNI).

Technical information about this service’s accessibility

The Department of Work and Pensions (DWP) is committed to making this service accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

This service is fully compliant with the Web Content Accessibility Guidelines version 2.2 AA standard.

Preparation of this accessibility statement

This statement was prepared on 10 January 2025. It was last reviewed on 20 March 2025.

This service was tested by the DWP Digital Accessibility Team and checked for compliance with WCAG 2.2AA.
